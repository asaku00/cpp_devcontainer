Repositorio creado para crear y utiliza un ambiente de desarrollo con Devcontainers.
Devcontainers necesita esencialmente Docker y Visual Code instalado con las exteniones de
Dev Containers
https://code.visualstudio.com/docs/devcontainers/containers

Una vez abierto el proyecto junto con docker correindo, con f1 en Visual abre el buscador de comandos
Dev Containers: Rebuild Container
Si es que no aparece como ventana emergenta la accion en el mismo Visual
